## ⚙️ Configuration Guides

Each Discord modification handles custom CSS and theme files in a slightly different manner. Follow the instructions for your chosen mod to ensure the themes load correctly.

### Vencord
Vencord keeps theme management settings inside the Discord window. Open your Discord User Settings and find the Vencord tab. Look for the Themes option. Paste the theme path or select your downloaded file here to enable the effect. You may need to click the refresh icon to see changes.

### BetterDiscord
BetterDiscord uses a dedicated Themes folder. Open your Discord User Settings and locate the BetterDiscord section. Click Open Themes Folder. Move your downloaded theme files into this folder. Return to Discord and toggle the theme on using the switch in the Themes menu.

### Vesktop
Vesktop functions similar to other Electron-based Discord apps. Go to the Vesktop settings menu. Navigate to the Themes section. Use the file picker to select your downloaded theme file. The app applies the visual changes once you select the file.

## 🪄 Adjusting Blur Presets

This loader includes several blur presets to change how transparent your background looks. If your background appears too opaque, open the theme settings menu. Select a lower blur value to increase transparency, or a higher value to make the background blurrier. 

The software updates the look in real-time. You do not need to restart Discord when you change these simple settings.

## 🛡️ Common Questions

### Does this violate Discord Terms of Service?
Modifying your Discord appearance is a common practice. Many users customize their clients to improve functionality or aesthetics. This software acts as a local skinning tool and does not interfere with Discord server data or account security.

### Why do I see a white background instead of transparency?
Ensure you have enabled hardware acceleration in your Discord settings. You can find this under the Advanced tab. Without hardware acceleration, your system cannot render the blur and transparency effects.

### How do I uninstall the themes?
Navigate to your theme manager within your Discord mod. Click Disable or Delete on the theme. The interface returns to the default Discord look immediately. You can remove the theme files from your computer once you delete them from the mod manager.

### Can I change the theme color?
The acrylic effect uses your system accent color by default. You can change this in your Windows Personalization settings. Changes made to your Windows accent color apply to the Discord theme automatically.

### What if the app fails to start?
Ensure you have the latest version of your Discord client installed. If the app detects an outdated client, it will stop the loading process to prevent errors. Check that you downloaded the latest release from the official repository link provided above. 

### Does this affect other applications?
No. The modifications remain strictly inside the Discord client. No other programs on your computer change their appearance. 

### Can I create my own blur presets?
The software allows manual editing of the underlying CSS files if you know basic web design, but the included presets cover most user needs. You can edit the theme file with any text editor to fine-tune the color intensity or blur radius if you want a custom look. 

## 📝 Support

If you run into issues, check your mod's documentation page. Issues usually stem from conflicting plugins or outdated versions of the Discord client. Keep your Discord app updated to maintain compatibility with the latest theme releases.
