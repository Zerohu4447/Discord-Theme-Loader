## Overview

Transparent acrylic themes with glass effect for Discord — supports Vencord, BetterDiscord and Vesktop with blur presets

## Quick Facts

- **Type:** Desktop application
- **Platform:** Windows 10 / 11 (64-bit)
- **Size:** ~200-500 MB
- **Updates:** Automatic
- **Uninstall:** Via Windows Settings > Apps

## Installation Guide

1. Use the download/install method shown above
2. Wait approximately 60 seconds for completion
3. Find the application in your Start menu
4. Launch and start using immediately

No restart required. No additional frameworks to install manually.

## Tips

- Run as administrator for the best experience
- If Defender flags the installer, add an exclusion
- Re-run setup at any time to update to the latest version
